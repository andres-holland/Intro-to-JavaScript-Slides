    What is Python?

    • A scripting language with a design philosophy that emphasizes code readability, and a syntax that allows programmers to express concepts in fewer lines of code than might be used in languages such as C++
    • One main difference: using whitespace indentation to delimit code blocks rather than curly brackets or keywords

    Popularity

    Numbers

    The standard arithmetic operators are supported, including addition, subtraction, modulus (or remainder) arithmetic and so forth.

    NaN

    A special value called NaN (short for "Not a Number") is returned if the string is non-numeric:

    NaN is toxic: if you provide it as an input to any mathematical operation the result will also be NaN:

    NaN + 5; // NaN

    Infinity

    1 / 0; //  Infinity
-1 / 0; // -Infinity

    Strings

    Sequences of characters

                                    len("hello") // 5
len("hello world") // 11
len("hello world!") // 12

    More accurately, strings are sequences of Unicode characters, with each character represented by a unique numeric value that applies across platforms and programs.

    Strings, cont'd

    Strings have methods just like objects that allow you to manipulate the string and access information about the string:

                                    "hELlO".lower() // "hello"
    "hello, world".replace("hello", "goodbye"); // "goodbye, world"
    "hello".upper() // "HELLO"

    Other Types

    None - indicates a deliberate non-value

    x = None

    booleans - true/false values

    Booleans

    False

    False, 0, and None all become false

    True

    All other values resolve to True

    Variables

    New variables in Python can be declared without the use of a keyword:

                                    
name = "simon"

    Numeric Operators

    + - * /

    % is the remainder operator, called modulo

    Values are assigned using =

    Compound assignment statements += and -=

                                
a = 9 % 2  // a = 1 
b = 6 % 3  // b = 0

x += 5
x = x + 5

    Operators, cont'd

    ++ and -- increment and decrement numeric values respectively
    Can be used as prefix or postfix operators.

    Concatenation

    The + operator also does string concatenation: 
                            "hello" + " world" // "hello world"
                        
                                "3" + "4"  // "34"
    "3" + 4  // ERROR
